# 开发日志

## 0.1.0

**发布时间**：2023.04.26

**简介**：alpha 版本发布。

**功能速览**：

- 所见即所得的 markdown 编辑
- 打开文件，打开文件夹
- 基础框架搭建
- 热更新
- 支持榕树功能，只支持榕图的展示功能，不支持榕图的编辑功能，完全不支持榕林功能
- 请不要随意用 ficus 打开重要文件，因为此版本存在一定风险清空用户的文件
